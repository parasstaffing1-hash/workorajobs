WorkoraJobs Technical SEO Audit & Site Health Monitoring Engine (Milestone 20)
=============================================================================

1. Purpose:
A production-grade, highly performant technical SEO audit engine designed to crawl, parse, evaluate, score, and monitor millions of programmatic pages on the WorkoraJobs site.

2. Core Features:
- Multi-Category Scoring Pipeline: Audits and ranks page health across Crawlability, Indexability, Metadata, Structured Data, Content Quality, Performance, URL Quality, and Internal Linking.
- Automatic Recommendation Engine: Compiles findings into prioritized fixes mapped to explicit remediation steps.
- Asynchronous BullMQ Queues: Processes resource-heavy crawls in the background with immediate 202 Accepted HTTP responses in the client APIs.
- Trend Analytics & Regression Detection: Tracks scoring indices historically, raising warnings if average health declines significantly.
- Automation Hooks: Integrates natively with n8n to trigger downstream alerts or automated fix workflows.

3. Database Models:
- TechnicalSeoAuditRun: Aggregates average category ratings, status, trigger types, and counts.
- TechnicalSeoPageAuditResult: Maps granular category health indices per page.
- TechnicalSeoIssue: Tracks individual issues, category mappings, severity indicators, and resolution statuses.
- TechnicalSeoRecommendation: Prioritized lists of programmatic fixes.

4. API Endpoint Index:
- POST /api/v1/seo-audit/trigger        : Queues full, incremental, or single-page crawls.
- GET  /api/v1/seo-audit/runs           : Paginated audit runs history.
- GET  /api/v1/seo-audit/runs/:id       : Granular results, recommendations, and issues.
- GET  /api/v1/seo-audit/trends         : Dynamic performance metrics trends over time.
- GET  /api/v1/seo-audit/issues         : Issue explorer index with search & resolution filtering.
- POST /api/v1/seo-audit/issues/:id/resolve : Resolves a specific issue flag.
- GET  /api/v1/seo-audit/export/:id     : Exports raw audit JSON file.
- GET  /api/v1/seo-audit/config         : Retrieves current rule weights and limits.
- PUT  /api/v1/seo-audit/config         : Updates active rule thresholds.
